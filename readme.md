Quiz-Generator

Summary: Upload a pdf of a lecture or script and let the website generate a quiz or exam with the help of ai.

Step 1: Upload a pdf file of a lecture, split it into topics.

Step 2: Let the user select which topics he wants to include

Quiz-Mode: . Create an XP-Bar for each topic that fills with each correctly answered Question. Randomly select a topic, generate a question and select a quiz type that fits the question. Quiz-types are: Single Choice, True/false, gap-text, free form text. After answering a question display a short explanation + the possibility of chatting with the ai. Flase answers get saved and repeated at a random time later. Correctly answered questions fill the xp bar of the corresponding topic. If all bars have been filled the user can select the exam mode.

Exam-Mode: The user gets displayed a number of questions at once, these will be free form questions. After handing in the exam, the ai grades the exam and rates it on a scale from 0 to 3 stars. Additionally a strength/weakness analysis is created that summarizes what topics the student should focus on based on the quiz and exam questions
